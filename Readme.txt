1. Install Maven at "https://maven.apache.org/install.html"(3.6.3)
2. Install IntelliJ Community 2020.1
3. Open project by IntelliJ
4. Open POM.xml file and refresh to download the library
5. Download Gecko Driver driver at "https://github.com/mozilla/geckodriver/releases/tag/v0.26.0" and copy to drivers folder in current project"
6. Open Login Test Under test > java > regression\
7. Right click on this and select "Run LoginTest"
